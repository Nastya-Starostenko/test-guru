Project Test guru!

App allows create and manage different tests

On the app can be two group of users: admin and user

User with role admin can:

  • Login into system
  • Create test
  • Update test

User with role user can:

  • Sign in/ sign up
  • Choose test
  • Pass test
  • Restart test
  • Can see the list of passed test

App has test and each test has list of questions with options for answer

There is a test project on the course from online-school Thinknetica
